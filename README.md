# School_Data

## Инструменты для реализации:
> 1) library - TeleBot, SQL3
> 2) SQL3 - id, class, day, time , lessons
> 3) Button - Одна копка все расписание, Вторя кнопка для конкретных дней и время
> 4) Button_teaxher - Кнопка для изменение расписание. 1 Восколько это (12:00)
                                                       2 Какой урок (Литература)
                                                       3 Какой день (Суббота)


## Проект состоит из:
> 1) TeleBot_commands - start, help (Раскажит всю информацию), button_schedule(Бутет разделение кенопок. " Одна копка все расписание, Вторя кнопка для конкретных дней и время")
> 2) TeleBot_commands_teaxher - button_teaxher (Бутет разделение кенопок. "Одна копка все расписание. Вторая кнопка для редактирование расписание ")
> 3) SQL3 - первая команда для расписаний
         вторая команда для редактирование " расписаний "
         третья команда для конкретных дней и временя " расписаний "
         четвертая команда для обновление всех расписаний в вреальном временни (Или хотябы раз в день)
